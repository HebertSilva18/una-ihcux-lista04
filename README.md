Prevenção vs. Tratamento

O try-catch é uma estrutura de tratamento de exceções presente em várias linguagens de programação.

O try-catch permite que o programa lide com erros em tempo de execução sem "quebrar" — ou seja, sem encerrar abruptamente.
 

Para um código robusto, use os dois juntos:

o Try age para prevenir o erro de acontecer
já o Catch executa caso um erro aconteça.
